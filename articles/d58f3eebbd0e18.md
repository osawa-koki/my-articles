## 前提

Windows ユーザが会社で Mac OS を使うことになったので、操作の違いをまとめました。  

| Windows | Mac OS |
| --- | --- |
| 11 Home | Ventura 13 |

## 操作比較

### $ ウィンドウの最大化

🪟🪟🪟 Windows  

```key
「Windowsキー」+「↑」キー
```

🍎🍎🍎 Mac OS

```key
「コマンドキー」+「コントロールキー」+「F」
```

### $ アクティブウィンドウの変更

🪟🪟🪟 Windows  

```key
「Windowsキー」+「Tabキー」
```

🍎🍎🍎 Mac OS  

```key  
「コマンドキー」+「Tabキー」
```

### コピー履歴の表示

🪟🪟🪟 Windows  

```key
「Windowsキー」+「Vキー」
```

🍎🍎🍎 Mac OS  

`Clipy(https://clipy-app.com/)`を使いましょう。  

### 画面ロック

🪟🪟🪟 Windows  

```key
「Windowsキー」+「Lキー」
```

🍎🍎🍎 Mac OS  

```key
「コマンドキー」+「コントロールキー」+「Qキー」
```

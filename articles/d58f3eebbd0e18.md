## 前提

Windowsユーザが会社でMacOSを使うことになったので、操作の違いをまとめました。  

| Windows | MacOS |
| --- | --- |
| 11 Home | Ventura 13 |

## 操作比較

### $ ウィンドウの最大化

🪟🪟🪟 Windows  

「Windowsキー」+「↑」キー  

🍎🍎🍎 MacOS

「コマンドキー」+「コントロールキー」+「F」  

### $ アクティブウィンドウの変更

🪟🪟🪟 Windows  

「Windowsキー」+「Tabキー」  

🍎🍎🍎 MacOS  

「コマンドキー」+「Tabキー」  

### コピー履歴の表示

🪟🪟🪟 Windows  

「Windowsキー」+「Vキー」  

🍎🍎🍎 MacOS  

`Clipy(https://clipy-app.com/)`を使いましょう。  

### 画面ロック

🪟🪟🪟 Windows  

「Windowsキー」+「Lキー」  

🍎🍎🍎 MacOS  

「コマンドキー」+「コントロールキー」+「Qキー」
